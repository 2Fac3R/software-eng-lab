# Project Context: Elixir Senior Engineer Preparation

## 1. Project Overview
This project is a comprehensive training plan for a **Senior Elixir Engineer** role. The plan is based on a job description emphasizing distributed systems, fault tolerance, and event-driven architectures.

## 2. Key Technologies & Concepts
- **Core Language:** Elixir and its functional programming paradigm.
- **Concurrency:** OTP (GenServer, Supervisor, Application) and the Actor Model.
- **Architecture:** Event-Driven Architecture (EDA), Domain-Driven Design (DDD), and Clean Architecture principles.
- **Messaging:** Apache Kafka for event streaming.
- **Database:** PostgreSQL with a focus on the Ecto library.
- **Best Practices:** TDD, code reviews, documentation, and ecosystem tools.

## 3. Module Structure
The training plan is divided into 6 modules, each with its own directory containing theory, challenges, solutions, cheatsheets, flashcards, and exams.

*   **`01_elixir_and_functional_programming/`**: Covers Elixir fundamentals, immutability, pattern matching, and functional concepts.
*   **`02_distributed_systems_with_otp/`**: Focuses on OTP, GenServers, Supervisors, and the "Let it Crash" philosophy.
*   **`03_event_driven_architecture_kafka/`**: Details EDA principles and integration with Kafka, particularly using Broadway.
*   **`04_advanced_software_design/`**: Applies SOLID principles to a functional context and covers architectural patterns.
*   **`05_relational_databases_and_ecto/`**: Covers advanced SQL and the idiomatic use of Ecto for database interactions.
*   **`06_technical_leadership_and_best_practices/`**: Focuses on non-coding skills like testing strategies, code reviews, and documentation.
