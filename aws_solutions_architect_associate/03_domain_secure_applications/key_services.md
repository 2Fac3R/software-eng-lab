# Domain 3: Key Services for Security

## IAM (Identity and Access Management)
*   **Users/Groups/Roles:** Role-based access control (RBAC).
*   **Policies:** JSON-based permissions.
*   **SCPs (Service Control Policies):** Define max permissions for an account in an AWS Organization.

## Amazon VPC (Security)
*   **VPC Flow Logs:** Capture IP traffic information.
*   **Security Groups:** Instance level firewall.
*   **NACLs:** Subnet level firewall.

## AWS KMS & CloudHSM
*   **KMS:** Key management, multi-tenant.
*   **CloudHSM:** Dedicated hardware module.

## AWS Secrets Manager
*   **Rotation:** Automatic rotation of database credentials.
*   **API integration:** Programmatic retrieval of secrets.

## AWS WAF & Shield
*   **WAF:** Protects against SQL injection, XSS (Layer 7).
*   **Shield Standard:** Free DDoS protection.
*   **Shield Advanced:** Managed 24/7 DDoS response.
