# Domain 3: Design Secure Applications and Architectures

La seguridad es la máxima prioridad en AWS.

## IAM (Identity and Access Management)
*   **Roles:** Otorgar permisos temporales a servicios.
*   **Políticas:** Documentos JSON que definen qué acciones están permitidas.

## Network Security
*   **VPC Security Groups:** Firewalls a nivel de instancia.
*   **NACLs:** Firewalls a nivel de subred.

## Data Protection
*   **AWS KMS:** Gestión de claves de cifrado.
*   **AWS Secrets Manager:** Gestión y rotación de credenciales de bases de datos.
