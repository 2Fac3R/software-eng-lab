# Domain 4: Key Services for Cost Optimization

## AWS Cost Explorer
*   **Analysis:** Visualize and manage your AWS costs and usage over time.
*   **Forecast:** Predict future spending based on historical data.

## AWS Budgets
*   **Alerts:** Receive notifications when costs or usage exceed (or are forecasted to exceed) your budgeted amount.

## AWS Trusted Advisor
*   **Recommendations:** Provides real-time guidance to help you provision your resources following AWS best practices (including cost).

## Compute Optimizer
*   **AI/ML:** Recommends optimal AWS resources for your workloads to reduce costs and improve performance.

## S3 Storage Classes
*   **Standard:** Active data.
*   **Standard-IA:** Infrequent access, minimum 30 days.
*   **One Zone-IA:** Infrequent access, single AZ.
*   **Glacier Instant Retrieval:** Archive data accessed in milliseconds.
*   **Glacier Deep Archive:** Archive data, 12-hour retrieval, lowest cost.
