# Domain 4: Design Cost-Optimized Architectures

Optimización de costes sin comprometer el rendimiento.

## Estrategias de Ahorro
*   **Right-sizing:** Ajustar el tamaño de las instancias a la carga real.
*   **Spot Instances:** Ahorros de hasta el 90% para cargas tolerantes a fallos.

## Gestión de Datos
*   **S3 Lifecycle Policies:** Mover datos antiguos a clases de almacenamiento más baratas (Glacier).
*   **S3 Intelligent-Tiering:** Gestión automática del coste de almacenamiento.
