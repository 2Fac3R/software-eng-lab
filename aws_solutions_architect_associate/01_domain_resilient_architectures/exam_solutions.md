# Domain 1 Exam Solutions

**Q1 Answer: B**
An Auto Scaling Group (ASG) across multiple Availability Zones (AZs) behind an Application Load Balancer (ALB) ensures that if one AZ fails, the instances in the other AZ(s) can still handle traffic.

**Q2 Answer: B**
Multi-AZ deployment provides synchronous replication, high availability, and automatic failover in the event of an AZ failure. (Read Replicas are primarily for scaling reads).

**Q3 Answer: A**
Amazon SQS (Simple Queue Service) provides decoupling by storing messages in a queue until they are processed by a consumer.
